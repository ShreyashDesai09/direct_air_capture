# direct_air_capture

Arduino-based direct air capture system for extracting CO₂ from ambient air and storing it in a tank.

## Overview

This repository contains the primary Arduino sketch for a direct air capture prototype. The project is intended to demonstrate CO₂ extraction from air using a microcontroller, sensors, and actuation components to move captured gas into a storage tank.

## Files

- `Mega_Project_Final_Code.ino` — main Arduino sketch for the direct air capture system.

## Setup

1. Open `Mega_Project_Final_Code.ino` in the Arduino IDE.
2. Connect the target Arduino board.
3. Select the correct board and serial port.
4. Upload the sketch.

## Hardware

Typical components for this project may include:

- Arduino board (Uno, Nano, Mega, etc.)
- CO₂ sensor
- Fans or pumps for air movement
- Valves and tubing
- Storage tank or reservoir

## Notes

- Verify sensor wiring and calibration before running the system.
- Ensure all electrical connections are secure and powered safely.
- Modify the sketch as needed for your specific hardware configuration.
